## Installation

`git clone {url}` this repo, then remove `.git` directory and install packages.

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the dev server at [http://localhost:3000](http://localhost:3000).

### Test server

For first start you should create a local repo

```bash
git init
```

Run tests

```bash
npm run test
```

Run TDD

```bash
npm run test:watch
```

### Production build

```bash
npm run build
```

> Note: `dist` directory.
